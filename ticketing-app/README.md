# Simple Ticketing System

A Python-based logic script that determines ticket pricing for a park based on the user's age. This project demonstrates the use of **conditional statements** (`if-elif-else`) and **comparison operators**.

### The Logic
The program categorizes users into three groups:
* **Full Ticket:** Age 18 and older.
* **Half Ticket:** Ages 6 to 17.
* **Free Entry:** Age 5 and younger.
